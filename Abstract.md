# EXcelの見方
+ 大カテゴリ
  + ラベルがどの大カテゴリに分類されるか
+ 話し手
  + どちらが話しているか
    + カ -> カウンセラー
    + ク -> クライアント
+ ラベル1
  + ラベル詳細のページにある「ラベル1」の方を入力する場所
+ ラベル2
  + ラベル詳細のページにある「ラベル2」の方を入力する場所
+ 本文
  + ラベルを付与する対象の文章

# ※注意点※(絶対に読んでください)
+ 大カテゴリの列のセルは絶対に内容を変更しないでください．
+ 本文と動画の両方を見てアノテーション（ラベル付）してください．
+ カウンセラーの行にはカウンセラーのラベルを，クライアントの行にはクライアントのラベルを付与してください．
